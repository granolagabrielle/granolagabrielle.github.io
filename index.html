About Me

I'm a software developer that is passionate about crafting applications that are human-centered and accessible.

In March of 2024, I decided to pivot from my career in higher education to the world of software development. I attended Emerging Digital Academy (EDA), where I learned to build full-stack applications using PostgreSQL, Express, React and Node.js.

I recently completed an internship as a front-end developer at Scheels, where I enhanced the functionality and accessibility of the Scheels mobile app for both iOS and Android using React Native and TypeScript. This experience really sparked my interest in building user-friendly and accessible experiences. Being able to put myself in the shoes of the user makes the work all the more engaging and fulfilling for me.

Before Scheels, I completed an internship at Bell Bank as a full-stack software engineer. I was able to round out my previous web dev heavy experience by learning about C#, .NET and Azure DevOps. Additionally, I learned about jQuery and Razor for creating dynamic front-end pages for .NET.

I am currently searching for a new opportunity where I can continue to build upon my skills as a developer. I'm a naturally creative person, and so front-end concepts tend to come more natually to me, but I enjoy the challenge of digging into back-end concepts and making sure software is secure, scalable and stable.
